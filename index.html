belajar css

Css adalah singkatan dari Cascading Style Sheet yang dipergunakan untuk menentukan tampilan dan format pada sebuah website. Artinya, dengan adanya CSS bisa mengatur mulai dari font, warna tulisan dan latar belakang pada website tersebut.

atribute font

font-family : mengatur jenis font
font-size : mengatur ukuran font
font-weight : mengatur ketebalan font
font-variant : mengubah font menjadi small caps
font-style : mengubah font menjadi cetak miring
line-height : mengatur spasi antar baris

atribute text

color : mengubah warna
text-align : mengatur format paragraf/text
text-indent : memberi indentasi atau tab
text-decoration : mengatur dekorasi underline,overline,dan line-trough
text-transform : mengubah jenis huruf menjadi huruf besar, kecil, dan kapital
letter-spacing : mengatur spasi antar huruf
word-spacing : mengatur spasi antar kata

background

background-color : mengatur warna background
background-image : mengatur gambar yang akan di gunakan pada background
background-position : mengatur gambar pada background
background-repeat : mengatur jenis pengulangan pada gambar

pseudo-class

:link : style defauld pada sebuah link
:hover : style ketika kursor berada di atas link/elemen
:active : style ketika link di klik
:visited : style ketika link sudah pernah dikunjungi
:first-child dan last-child : memilih anak elemen pertama atau terakhir
:Nth-Child(n) : memilih elemen ke n (1,2,3.../2n, 3n+2, 2n-1 .../ even & odd)
:target : ketika class id (#selektor) di target
:first-of-type : memilih elemen pertama berdasarkan jenisnya
:last-of-type : memilih elemen terakhir berdasarkan jenisnya
:Nth-0f-Type(n) : memilih elemen ke (n) berdasarkan jenisnya
