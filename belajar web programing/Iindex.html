css layouting

display : inline,blok, inline-blok
dimensi : width,height
overflow : visible, auto, hidden,scrol
box model : margin, padding, border, content

Overlaping margin secara vertikal akan mengambil ukuran tebesar, sebaliknya jika horisontal akan menjumlahkan kedua marginnya
float : none, left, right

Dengan memberi float, akan membuat nya keluar dari float normal dan akan di anggap tidak ada oleh elemen yang berada pada flow normal

elemen yang di beri position selain statik dapat menggunakan properti top, left, bottom, right untuk mengatur posisinya.
relativ : akan memindahkan elemen ke dimensi lain, tetapi tempat awalnya nmasih di anggap ada dan relativ terhadap posisi awalnya
absolute : akan menaganggap elemen sebelumnya hilang dan relatif terhadap elemen pembungkusnya
fixed : akan relativ terhadap window, dan akan terkunci di posisi yang di tentukan
z-index : untuk menentukan tumpukan terhadap sumbu z elemen yang di beri posisi non statik
