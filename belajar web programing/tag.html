HTML DASAR

tag

text : H1-H6, dan p, dll
pendukung text : br, hr(horisontal), em, strong, dll
list(bullets & numbering) : ul (type: 1,A,I,i), ol (type: disk, square, circle), li, dl(dt dan dd)
tabel : table, thead,tbody
form : form, input, select, button
tag lainnya : script, object

atribut

acceskey : untuk menentukan elemen bisa di akses keybord
class & id : untuk memberi tanda pada elemen
dir : direction untuk menentukan arah tulisan
lang : untuk menentukan bahasa
style : untuk menyisipkan inline css
tabindex : untuk menentukan urutan tab
title : untuk memberi judul

table

tr : tabel row / baris
td : tabel data
thead : untuk head
tbody : untuk body
